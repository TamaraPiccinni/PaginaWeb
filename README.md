# Proyecto: Página Web "Mejor y Más Linda"

Este es un proyecto educativo diseñado para enseñar prácticas modernas de desarrollo web (HTML5 y CSS3).

El objetivo es mostrar la evolución desde los estilos en línea (`<p style=...`) y las malas prácticas que se enseñaban antiguamente, hacia un enfoque limpio, semántico, responsivo y fácil de mantener.

## Temas Cubiertos

<details>
<summary>Haz clic aquí para ver todos los temas (Temas 1-5 + Bono)</summary>

* **Tema 1: HTML Semántico**

  * La importancia de usar `<header>`, `<main>`, `<section>`, y `<footer>` en lugar de `<div>` para todo.

  * Beneficios para el SEO y la accesibilidad.

* **Tema 2: Responsive Design**

  * El uso obligatorio de la etiqueta `<meta name="viewport">`.

  * Fundamentos de las **Media Queries** (`@media`) para adaptar el diseño a celulares.

* **Tema 3: Alineación con Flexbox**

  * El método moderno para alinear y centrar elementos, reemplazando técnicas antiguas.

  * Demo interactiva de `display: flex`, `justify-content` y `align-items`.

* **Tema 4: Estética y CSS Moderno**

  * Uso de **Variables CSS** (`:root` y `var(--color)`) para crear temas y facilitar el mantenimiento.

  * Importar fuentes personalizadas desde Google Fonts.

  * La importancia del espaciado (Padding y Margin).

* **Bono Extra: "Magia" CSS**

  * Efectos de interacción con `:hover`.

  * Animaciones suaves con `transition`.

  * Profunidad y elevación usando `box-shadow`.

* **Tema 5: Estructura y Deploy**

  * La estructura de carpetas estándar (`/styles`, `/images`, etc.).

  * Cómo hacer "deploy" (publicar) un sitio estático gratis usando **Netlify Drop** y **GitHub Pages**.

* **Funcionalidad Adicional: Modo Oscuro/Claro**

  * Un interruptor funcional que usa JavaScript para cambiar la clase en la etiqueta `<html>` y alternar las variables CSS.

</details>

## Tecnologías Utilizadas

Este proyecto está construido 100% con las tecnologías base de la web:

* **HTML5** (para una estructura semántica)

* **CSS3** (para todos los estilos, usando Variables, Flexbox, y Media Queries)

* **JavaScript (ES6+)** (una pequeña sección, muy comentada, para la funcionalidad del botón de tema)
